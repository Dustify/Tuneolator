# Tuneolator

A wee wavetable synth for STM32F103Cx boards.

DAC provided by 8-bit R-2R ladder on GPIO port A pins 0-7.

#### Features
- [x] Polyphony (10+ capable)
- [x] Sine wave
- [x] Triangle wave
- [x] Sawtooth wave
- [x] Velocity
- [x] LFO
- [x] LED heartbeat
- [x] Mario demo
- [x] Axel F demo
- [x] LFO
- [x] 48KHz update rate
- [x] 5k point wave resolution
- [x] Split keyboard

- [ ] ADSR
- [ ] Square wave
- [ ] MIDI interface
- [ ] 'Smart' compression
