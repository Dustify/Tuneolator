# Tuneolator

A wee wavetable synth for STM32F103Cx boards.

DAC provided by 8-bit R-2R ladder on GPIO port A pins 0-7.

#### Features
- [x] Polyphony
- [x] Sine wave
- [x] Triangle wave
- [x] Sawtooth wave
- [x] Square wave
- [x] Velocity
- [x] LED heartbeat
- [x] Mario & Axel F demos
- [x] LFO (performance issues with polyphony, ADSR)
- [x] 48KHz update rate
- [x] 7k point wave resolution
- [x] Split keyboard
- [x] Note on indicator
- [x] MIDI interface

- [ ] ADSR
- [ ] 'Smart' compression
- [ ] Drums?
- [ ] Pitch bending
- [ ] 'Busy' indicator
